The HIMARPL Admin Panel is available at [admin.himarpl.com](https://admin.himarpl.com) with the admin role. Built with NextJS, this platform is exclusively accessible to HIMARPL members through a private repository.

:::note[New Staff Members]
  Please review the [staff guide](/getting-started/04-staff-guide) before getting started.
:::

## Core Features

Our admin panel offers three main functionalities:
- Member Management: Add, update, and oversee HIMARPL membership
- Department Administration: Handle HIMARPL's organizational structure
- Analytics Dashboard: Track real-time user activity and monitor content creation trends
