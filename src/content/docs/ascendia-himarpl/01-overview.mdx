[ascendia.himarpl.org](https://ascendia.himarpl.org) is the official website for Ascendia Cabinet, Himpunan Mahasiswa Rekayasa Perangkat Lunak (HIMARPL) at The Education University of Indonesia. The website is built using Next.js with the T3 Stack and is deployed on Vercel.

## Key Features

- Important Dedication Information
- Organization Structure Visualization
- News & Updates Integration
- Department Information Pages
- Interactive UI Components
- Responsive Design
- Smooth Animations
