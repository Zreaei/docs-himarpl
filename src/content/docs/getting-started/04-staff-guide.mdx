## Your Key Responsibilities

As a staff member in Kominfo, you'll be responsible for:

1. **Service Availability**
   - Monitor website uptime
   - Handle technical issues promptly
   - Ensure all services are running smoothly

2. **Access & Security**
   - Manage user access levels
   - Keep credentials secure
   - Monitor for security issues

3. **Regular Updates**
   - Deploy website updates
   - Update content when needed
   - Keep dependencies up to date

4. **Development Coordination**
   - Work with other developers
   - Review code changes
   - Coordinate feature releases

5. **Documentation**
   - Keep documentation current
   - Document system changes
   - Record technical decisions

## Input New Staff Data

Follow these steps to add new staff members:

1. **Access the Admin Panel**
   - Go to [admin.himarpl.com](https://admin.himarpl.com)
   - Login using the main HIMARPL account

2. **Navigate to Staff Section**
   - Go to the `Staff` section
   - Change the period to the current year (e.g., 2025)

:::note[NOTE]
If there is no data for the current period, you can add it manually using prisma studio.
:::

3. **Prepare Staff Images**
   - Upload staff photos to the [assets-himarpl](https://github.com/himarpl/assets-himarpl) repository
   - Convert GitHub links to CDN links using [jsdelivr](https://www.jsdelivr.com/github)

4. **Add Staff Members**
   - Click "Add New Staff"
   - Fill in staff details
   - Paste the jsdelivr CDN link in the `image` field
   - Save the changes

:::note[TIP]
You can reference previous period's staff data to see examples of properly formatted entries.
:::
