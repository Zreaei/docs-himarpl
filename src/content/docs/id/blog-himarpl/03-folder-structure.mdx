## Penjelasan Struktur Direktori

Proyek ini mengikuti arsitektur [T3 Stack](https://create.t3.gg/) dengan [TipTap](https://tiptap.dev/) untuk rich text editing. Struktur ini menekankan type-safety, modularitas, dan pemisahan yang jelas antar konsep.

### Direktori Root

- **prisma/**: Berisi file yang berkaitan dengan database menggunakan Prisma ORM
  - `schema.prisma`: Mendefinisikan model database, relasi, dan konfigurasi
  - `seed.js`: Script seeding data awal
- **public/**: Aset statis yang disajikan langsung
  - Gambar, font, dan sumber daya publik lainnya
- **src/**: Direktori source code utama yang berisi semua kode aplikasi

### Direktori App (Next.js App Router)

Direktori `app/` mengikuti konvensi Next.js 14 App Router:

- **api/**: Rute API backend
  - `trpc/`: Konfigurasi dan endpoint API tRPC
- **Halaman berbasis fitur**: Setiap fitur utama memiliki direktori sendiri
  - `me/`, `search/`, `new/`, dll.
  - Masing-masing berisi `page.tsx` untuk tampilan utama
- **Rute Dinamis**: Menggunakan routing berbasis file Next.js
  - `[username]/`: Rute dinamis untuk profil pengguna
  - `tag/[tag-slug]/`: Rute dinamis untuk halaman tag

### Direktori Components

Komponen diorganisir berdasarkan fitur dan tanggung jawab:

- **Komponen Fitur**: Dikelompokkan berdasarkan fitur
  - `editor/`: Komponen rich text editor
  - `home/`: Komponen khusus halaman beranda
  - `post/`: Komponen yang berkaitan dengan post
  - `search/`: Komponen interface pencarian
  
- **Komponen Umum**: Dibagikan lintas fitur
  - `common/`: Komponen global seperti navbar dan footer
  - `brand/`: Komponen yang berkaitan dengan branding
  - `ui/`: Komponen UI yang dapat digunakan kembali (menggunakan shadcn/ui)
    - Button, form, modal, dll.
    - Konfigurasi tema dan komponen dasar

### Direktori Inti

- **hooks/**: Custom React hooks untuk logika yang dibagikan
- **lib/**: Utilitas inti dan logika yang dibagikan
  - Utilitas umum
  - Definisi tipe
- **server/**: Kode khusus server-side
  - Implementasi API
  - Setup otentikasi
  - Konfigurasi database
- **trpc/**: Konfigurasi dan router tRPC
  - Definisi rute API
  - Prosedur type-safe

### Styles

- **styles/**: Konfigurasi styling global
  - `globals.css`: Style CSS global, konfigurasi Tailwind, dan konfigurasi tema Shadcn/UI


## Mengimpor File

Untuk mengimpor file, Anda dapat menggunakan alias `@/`. Alias ini secara otomatis dikonfigurasi dalam proyek.

```tsx title="src/app/page.tsx"
import { Navbar } from "@/components/common/navbar";
import { Articles } from "@/components/home/articles";
import { Sidebar } from "@/components/home/sidebar";
import { Button } from "@/components/ui/button";
import { Sheet, SheetContent, SheetTrigger } from "@/components/ui/sheet";
import { getServerAuthSession } from "@/server/auth";
```

Pendekatan ini sangat berguna ketika Anda bekerja pada proyek besar dengan banyak file dan folder dan dapat dilakukan dengan memperbarui file tsconfig.json sebagai berikut:
