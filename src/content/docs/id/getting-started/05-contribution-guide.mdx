Sebelum berkontribusi, harap baca:

- Kode Etik
- Pedoman Kontribusi
- Sertifikat Asal Pengembang 1.1

di setiap repositori yang akan Anda kontribusikan di bagian [proyek open-source](/id/getting-started/03-opensource-projects).

## Pedoman Commit

- Ikuti spesifikasi Conventional Commits
- Tanda tangani commit Anda (direkomendasikan)
- Pesan commit tidak boleh diakhiri dengan titik
- Sertakan tipe yang sesuai (feat, fix, docs, dll.)

## Kebijakan Pull Request

PR harus memenuhi persyaratan ini sebelum digabungkan:

- Semua pemeriksaan status yang diperlukan harus lulus
- Semua diskusi harus diselesaikan
- Beberapa commit harus digabungkan (squash)
- Ikuti template PR

license (unless I am permitted to submit under a different license), as indicated in the file; or

of the contribution (including all personal information I submit with it, including my sign-off)
