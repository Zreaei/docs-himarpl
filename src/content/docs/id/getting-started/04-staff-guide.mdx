Selamat datang di departemen Kominfo! Panduan ini akan membantu Anda memahami tanggung jawab dan sistem yang akan Anda kelola.

## Tanggung Jawab Utama

Sebagai staff Kominfo, Anda bertanggung jawab untuk:

1. **Ketersediaan Layanan**

   - Monitor uptime website
   - Tangani masalah teknis dengan cepat
   - Pastikan semua layanan berjalan lancar

2. **Keamanan & Akses**

   - Kelola tingkat akses pengguna
   - Jaga keamanan kredensial
   - Monitor isu keamanan

3. **Pembaruan Rutin**

   - Deploy pembaruan website
   - Update konten saat diperlukan
   - Jaga dependensi tetap up-to-date

4. **Koordinasi Pengembangan**

   - Kerja sama dengan developer lain
   - Review kode
   - Koordinasi rilis fitur

5. **Dokumentasi**
   - Jaga dokumentasi tetap aktual
   - Dokumentasikan perubahan sistem
   - Catat keputusan teknis

## Prosedur Harian

### 1. Pemeliharaan Rutin

- Backup mingguan
- Cek log sistem
- Monitor error
- Update konten sesuai kebutuhan

### 2. Pengelolaan Masalah

- Monitor GitHub issues
- Respons laporan pengguna
- Koordinasi perbaikan bug
- Tracking permintaan fitur

### 3. Komunikasi Tim

- Hadir rapat tim
- Update laporan progres
- Koordinasi dengan departemen lain
- Dokumentasi keputusan
