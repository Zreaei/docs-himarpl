HIMARPL mengelola beberapa proyek open-source yang dapat dikontribusikan oleh siapa saja:

- [ascendia-himarpl](https://github.com/himarplupi/ascendia-himarpl) - Website utama kabinet HIMARPL saat ini
- [blog-himarpl](https://github.com/himarplupi/blog-himarpl) - Platform blog pengurus HIMARPL
- [bot-himarpl](https://github.com/himarplupi/bot-himarpl) - Bot telegram HIMARPL (Phoenix-chan)
- [pmb-himarpl](https://github.com/himarplupi/pmb-himarpl) - Portal Informasi Mahasiswa Baru
- [mabim-himarpl](https://github.com/himarplupi/mabim-himarpl) - Website Masa Bimbingan Rekayasa Perangkat Lunak 2024
- [docs-himarpl](https://github.com/himarplupi/docs-himarpl) - Dokumentasi teknis (website ini)
