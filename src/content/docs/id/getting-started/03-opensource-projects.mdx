HIMARPL memiliki beberapa proyek open-source yang dapat dikontribusikan oleh siapapun:

1. www-himarpl - Website utama HIMARPL
2. blog-himarpl - Platform blog
3. pmb-himarpl - Sistem Penerimaan Mahasiswa Baru
4. mabim-himarpl - Portal Masa Bimbingan
5. docs-himarpl - Dokumentasi teknis (website ini)
