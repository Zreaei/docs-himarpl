title: Pertimbangan Masa Depan
description: Memandang ke depan - apa selanjutnya untuk website PMB HIMARPL?
---

Mari kita bicara tentang beberapa hal yang harus dipikirkan oleh kabinet PMB HIMARPL masa depan ketika bekerja untuk meningkatkan website PMB HIMARPL.

## Apa yang Kami Pelajari

- Koordinasi dengan departemen Advokastra sangat penting. Website saat ini dibangun tanpa koordinasi dengan departemen Advokastra, yang berarti bisa terlihat dan terasa jauh lebih baik. Kabinet masa depan harus mencoba melibatkan desainer di Departemen Advokastra untuk membuat situs lebih user-friendly dan visual yang luar biasa.
- Menambahkan konten secara manual ke MDX di repositori github assets-himarpl agak merepotkan, ini tidak terintegrasi dengan blog karena dibuat sebelum blog dibuat.

## Beberapa Ide untuk Dipertimbangkan

- Di kabinet selanjutnya, coba ubah informasi terpenting di website, seperti informasi pendaftaran, agar lebih relevan dengan tahun saat ini. Juga, informasi kontak harus diperbarui ke anggota Advokastra saat ini.
- Coba dorong Staf Advokastra untuk membuat artikel berita di [blog.himarpl.com](https://blog.himarpl.com) dan tag artikel berita dengan `berita pmb`, `snbt`, `snbp`, `pmb`, `prestasi istimewa`, `sm upi`. Ini akan membantu website mendapatkan lebih banyak traffic dari mesin pencari, terutama untuk kata kunci terkait `pmb himarpl`.
- Coba buat website lebih user-friendly, seperti menambahkan search bar ke website, atau menambahkan fitur ke website untuk memudahkan menemukan informasi yang dibutuhkan.
- Migrasi dan integrasikan semua konten baru dari MDX di repositori github assets-himarpl ke blog himarpl dengan tag atau label tertentu (seperti snbp, snbt, pmb, prestasi istimewa, sm upi), ini akan membuat website lebih up to date dan lebih mudah dikelola.
